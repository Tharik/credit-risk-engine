# 🤖 AI Journey

This folder documents how AI tools were used throughout the development of the Credit Risk Engine challenge.

The objective was not to outsource engineering decisions, but to accelerate execution while maintaining full ownership of:

* Architecture
* Design quality
* Validation
* Testing
* Trade-off decisions

---

# 🛠️ Tools Used

* ChatGPT
* GitHub Copilot
* VS Code IntelliSense

---

# 🎯 Primary AI Use Cases

## Used for:

* Initial architectural brainstorming
* Rule engine design refinement
* Test scenario generation
* Docker hardening improvements
* CI/CD workflow generation
* README structuring
* Validation strategy suggestions

---

## NOT used blindly for:

* Final business logic acceptance
* Rule correctness verification
* Edge case decisions
* Security decisions
* Design trade-offs

All AI-generated outputs were manually reviewed and iterated upon.
